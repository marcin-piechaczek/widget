### About

Simple widget project with HTML/CSS/jQuery

This app uses:
* jQuery v3.5.1,
* Block Element Modifier methodology
* SVG animation

Files structure:
```bash
├── assets
│   ├── app.css
│   ├── app.js
│   ├── chevron-down-solid.svg
│   └── location-arrow-solid.svg
├── index.html
└── readme.md
```
